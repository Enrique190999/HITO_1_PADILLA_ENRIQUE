1. Explica que tipado (débil o fuerte) se aplica en JavaScript


JavaScript es un lenguaje con un tipado débil ya que las variables se declaran sin explicar el tipo que son (numericas, booleanas, texto...), los datos almacenados en ellas se pueden trabajar sin tener que convertirlas (solo en casos muy concretos).

3. Explica la diferencia entre let, var y const con un ejemplo

Tenemos diferentes tipos de variables segun la necesidad que tengamos:

var

Son variables de alcance local las cuales, a diferencia de variables const si pueden cambiar su valor

Estas variables tienen una peculiaridad y es que se pueden instanciar con el mismo nombre varias veces sin dar un error



let

Es un tipo de variable que si puede cambiar su valor, pero solo vivira en el bloque de fue creada


const

Es un tipo de variable que JAMAS cambiara su valor en ningun momento
