Ejercicio 2 - DOM vs BOM en JavaScript


1. Diseña una aplicación web con un formulario, imagenes y tablas.









Introduzca los datos y pulse en Enviar para continuar


2. Modifica los contenidos de cada uno de los elementos utilizando JavaScript mediante acceso al Document Object Model


En este formulario puedes cambiar cualquier valor que hayas introducido anteriormente unicamente escribiendolo y dando a Modificar datos

Al finalizar se mostraran los datos en el formulario principal

(Al pulsar en modificar datos muestra el siguiente apartado del ejercicio)


3. Realizar al menos 3 ejemplos de manejo de BOM. Por ejemplo, identificar navegador, consultar dimensiones de la ventana, mostrar listado de paginas visitadas...


(Al pulsar en el botón crea un elemento por JavaScript mostrando los valores justo debajo del botón)
